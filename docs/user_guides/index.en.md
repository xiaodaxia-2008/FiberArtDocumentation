# FiberArt Automated Fiber Placement Software User Documentation

This documentation provides guidance for using the `FiberArt` software. Users are encouraged to follow the tutorial steps while performing the software operations.

!!! attention "Attention"
    This documentation applies to **FiberArt version 0.14.3**. Features in other software versions may differ from those described here. 
    Please contact us promptly to upgrade to the latest version. 
    If you find any discrepancies, please [contact us](mailto:fiberart@duck.com) so we can update the documentation.

Please [download sample model data](https://gitee.com/zen_shawn/FiberArtData/repository/archive/master.zip) first, or [visit here](https://gitee.com/zen_shawn/FiberArtData) to download as needed.
